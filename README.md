
# Vehicle Rental Management System

A user can rent vehicle in online using this api. This is a backend api for rental management.

## Features

- user can signup and signin 
- authorization & authentication for user ("admin",  "customer")
- admin can access all the routes
- customer have limited access in this api
- user can rent a vehicle

## Technology Stack

- Node.js + TypeScript
- Express.js (web framework)
- PostgreSQL (database)
- bcrypt (password hashing)
- jsonwebtoken (JWT authentication)
## Installation

First you have to clone my project in your pc for installing project. then run those command in your pc command line.

```bash
  npm install
```
    
### Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`PORT`

`CONNECTION_STRING`

`SECRET`
